# wukong-itchat

用于微信控制 [wukong-robot](http://github.com/wzpan/wukong-robot.git) 。

## 安装

1. 先安装 wukong-robot 并运行，确保 wukong-robot 的版本 >= 1.5；
2. 安装依赖：

``` bash
pip3 install -r requirements.txt
```

## 运行

``` bash
python3 bot.py
```

## 配置

详见 config.yml 。

## 使用

与文件传输助手聊天，文本或者语音都将会作为指令转发给 wukong-robot 的后台管理端。
